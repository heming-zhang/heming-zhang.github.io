Deep Signaling Syngergy

 

A novel deep learning model to predict drugs combination effect and ananlyse critical signaling pathways

Using the number of cell line genes and signaling pathways as nodes number for 1st and 2nd hidden layer, we construct a not fully connected neural network between these two layers to analyse the critical pathways and predict the corresponding drugs combinations synergistic scores.
